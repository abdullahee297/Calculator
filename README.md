Web Calculator

This is a simple web-based calculator project.
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

What this project does

Takes input using on-screen buttons

Displays numbers and operators on the screen

Calculates the result when = is pressed

Clears input (C) or removes the last digit (X)

Shows an error for invalid expressions

Languages & Tools Used

HTML – creates the structure of the calculator

CSS – styles the calculator and makes it look good

JavaScript – adds calculator logic and functionality

Bootstrap – helps with responsive design
